About me

Programming and problem solving has always somehow accompanied my life, both privately and professionally. Probably around of 2008, I started to create my first batch scripts and I gained my first experience in the web by exploring the code of web pages. Beginning in 2010 I started to become a electrician in building automation where I could build up a great knowledge in programming PLC's and micro-controllers. In 2013 I decided to try my hand at creating my first website for a club and slowly I tumbled head first into the rabbit hole of coding and development. However back then my love for web development had not fully developed.

Fast-forward to 2019, on my job in tech support, I had the opportunity to again focus on web based technologies, while creating software components and extension for some products of our company. And it should not take long until I was fully in love with it. So, I started to dig in deeper and gain experience by creating some little front-end, back-end and web based desktop applications.

In 2024 I had to search for new opportunities, because my company had to close down the location where I worked. So I decided to take the chance and start with a new chapter in my life in the field of web development. As this seems to be the right way, based on my experience and my passion for it and also in terms of personal development.

Some Things I've Built

Redesign of the Website from MCH @ My Own

Code @ GitHub
November 2023 - Today
TypeScript - Node.js - Express.js - API - Vue/Pinia - SQL

This project is focused on a complete redesign of the website from mch-huelben.de, which I made with PHP back in 2013. The site was no longer state of the art and the idea was to level up this project using lot of skills I gain over time and also exploring new ones.

This project runs on a node/express HTTP server with a MariaDB Database using typeorm. Site rendering is done using Vue 3 and Pinia. The project is fully able to render on server side and is able to hydrate the rendering state and render additional pages/content directly on client side. This overall improves the performance by a lot, makes it light weight and can also be easily used for administration purposes.

So I created accidentally a little SSR/CSR SPA framework.

readlines-iconv @ My Own

Code @ GitHub
December 2023 - January 2024
JavaScript - TypeScript - Node.js - NPM

I had to read in multiple files with various encodings, fast and easily - So I came up with my own solution.

A small little npm module that is able to read files line by line synchronously or asynchronously. It includes per default a evaluation of the end of line character of the file (CRLF, LF, CR) and supports a lot of possible encodings thankfully to the integrated iconv-lite module. It makes use of the JavaScript Iterator Protocol to conveniently deliver line by line.

n4-simple-build @ My Own

Code @ GitHub
May 2023 - January 2024
JavaScript - TypeScript - Node.js - NPM - Niagara Supervisor

Node.JS/NPM-module to build simple N4 Framework modules via CLI. The goal was that it is simple to use without the need for an IDE or lots of deeper experience with Java or N4. For simple resource modules which do not need java files or which not require a java compiler.

It is also able to extract original (english) lexicon files (language specific text files) from N4 java modules and can create CSV-files for translation into different languages. It can re-import translated CSV-files for building simple Lexicon modules for N4.

Change Version Toolkit @ SBC Deutschland GmbH

January 2023 - February 2024
JavaScript - TypeScript - Electron.js - Node.js - Niagara Supervisor

A small and simple Electron.js based desktop application designed to switch and manage different versions of N4. In engineering with N4 it is often necessary to switch between different versions and to access different tools, folders and settings of N4. This application is designed to easily support the handling of N4 in different versions.

It can install/uninstall, detect, monitor and manage the state of the Windows based service daemon for different versions and installations. It features hotkeys for the most commonly used features and directories. It also features an supportive and easy way to adjust the heap of the Java runtime of N4 which is always specifically necessary depending on the project and available resources.

Enhanced Web Viewer @ SBC Deutschland GmbH

December 2022 - February 2024
JavaScript - TypeScript - Electron.js - Node.js - HTTP

Electron.js based desktop application for Windows and Linux. It is designed to act like a browser with additional functionality. It is manly designed to manage and visualize the web-based application of SBC's PLC-Controllers. There are limited possibilities on older PLC-Platform which required a client side app which could handle some functionality over different origins.
It can interact with that application to store specific persistent data between sessions and to share such data between different PLC-Controllers, to enhance the overall functionality. It has a additional alarming functionality (visual and acoustic), to inform the user in case something is wrong on the plant. It is capable for caching sessions which heavily reduces loading times over slow Ethernet connections.

Advanced Menu @ SBC Deutschland GmbH

December 2019 - March 2024
JavaScript - TypeScript - Niagara Supervisor - Handlebars - JSON

Is a fully customizable and dynamic frontend dashboard with an integrated customizable menu.

It is possible to open up any possible view from within N4 inside of the main frame of the Advanced Menu. The Advanced Menu can connect to different data APIs from N4 to show up alarm information, specific boolean or numeric data information and weather information (e.g. using 3rd party APIs). The Advanced Menu features a configurator tool to easily configurate the full functionality using JSON.

Parts of the configuration and the language of the Advanced Menu can be differently configured for each user. The Advanced Menu also has a fully configurable theme for different brandings in dark and light mode.
