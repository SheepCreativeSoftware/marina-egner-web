About me

Programming and problem solving has always somehow accompanied my life, both privately and professionally. Probably around of 2008, I started to create my first batch scripts and I gained my first experience in the web by exploring the code of web pages. Beginning in 2010 I started to become a electrician in building automation where I could build up a great knowledge in programming PLC's and micro-controllers. In 2013 I decided to try my hand at creating my first website and slowly I tumbled head first into the rabbit hole of coding and development. However back then my love for web development has not yet fully developed.

Fast-forward to 2019, on my current job in tech support, I had the opportunity to again focus on web based technologies, while creating software components and extension for some products of our company. And it should not take long until I was fully in love with it. So, I started to dig in deeper and gain experience by creating front-end, back-end and web based desktop applications.

Here are a few technologies I've been working with recently:

  • JavaScript (ES6+)
  • Node.js
  • Express.js
  • TypeScript
  • Electron.js
  • SQL

Some Things I've Built

n4-simple-build @ My Own

Code @ GitHub
Juli 2020 - Today
JavaScript - TypeScript - Node.js - NPM - Niagara Supervisor

Node.JS/NPM-module to build simple N4 Framework modules via CLI. The goal was that it is simple to use without the need for an IDE or lots of deeper experience with Java or N4. For simple resource modules which do not need java files or which not require a java compiler.

It is also able to extract original (english) lexicon files (language specific text files) from N4 java modules and can create CSV-files for translation into different languages. It can re-import translated CSV-files for building simple Lexicon modules for N4.
